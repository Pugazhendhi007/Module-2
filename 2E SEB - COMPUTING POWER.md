### AIM  
To write a Python Program to count the number of digits in a number.



---

### ALGORITHM


1. **Start**
2. **Input** the number `n`.
3. **Initialize** `count = 0`.
4. **Repeat** while `n > 0`:
   - Increase `count` by 1.
   - Divide `n` by 10 (discard the remainder).
5. **Print** the value of `count`.
6. **End**
